Lexer Safety: The Golden Rules

Custom Range-Keys (e.g., q1, morning) must adhere to these constraints to prevent lexer collisions:

  • No Control Characters: > < + = , . ! @ # $ % ^ & * ( ) [ ] { } are reserved.
  • No Leading Numbers: 1q is invalid. Leading digits are reserved for repeat counts.

Engine Modifiers

Modifiers control the "search vector" and "boundary alignment" of the resolution loop.

Modifier Type Meaning Example
> Forward Jump to next boundary #qtr.>
< Backward Jump to previous boundary #qtr.<
+ Relative (+) Offset forward by N units #qtr.+2
- Relative (-) Offset backward by N units #qtr.-1
next Alias (>) Semantic forward skip #qtr.next
this Identity Snap to current boundary #qtr.this

Primary Methods

.set()

The Anchor. Hard-aligns the instance to the specified term start.

t.set('#qtr.q1')

.add()

The Momentum. Advances the cursor relative to its current position.

t.add('#qtr.>2')

.until()

The Future Goal. Measures duration from current point forward to target.

t.until('#qtr.>')

.since()

The Past Origin. Measures duration from target up to current point.

t.since('#qtr.<')

Bifurcated Logic

Deterministic Sorting

The engine selects targets based on the présence of a directional modifier:

  • Absolute Search: Finding #qtr.q1 uses proximity sorting. It snaps to the Q1 physically closest to the current point.
  • Directed Search: Finding #qtr.>q1 uses chronological filtering. It only considers candidates starting strictly after the current point.

Practical Implementation

// Resolving a fiscal quarter shift in Sydney
const t = new Tempo('2025-12-25', { sphere: 'south' });

// Move exactly 2 quarters forward
t.add('#qtr.>2');

// Calculate time remaining until next Q1
const remaining = t.until('#qtr.>q1');
console.log(remaining.toString());
